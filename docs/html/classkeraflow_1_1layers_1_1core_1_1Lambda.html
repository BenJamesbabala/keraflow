Detailed Description

Wrapper for implementating simple inline layer.

  • input_shapes: nD, (nb_samples, x, y, ...)
  • output_shape: nD, specified by output_shape argument.

Examples

```python

add a x -> x^2 layer

model.add(Lambda(lambda x, factor: x ** 2)) ```
