Detailed Description

Applies Dropout to the input.

In training phase, for each unit in the input tensor, Dropout sets it to 0 with probability drop_rate and sets it to 1/(1-drop_rate) times with probability 1-drop_rate. In testing phase, Dropout simply returns the input tensor.

  • input_shape: nD, (nb_samples, ...)
  • output_shape: nD, (nb_samples, ...)
  • Reference: Dropout: A Simple Way to Prevent Neural Networks from Overfitting
